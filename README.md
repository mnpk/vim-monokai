# vim-monokai

Monokai color scheme for Vim.

## Screenshot

![Monokai in Vim](http://i.imgur.com/zuiElX2.png)

## Installation
##### Vim-Plug (https://github.com/junegunn/vim-plug)

```
Plug 'mnpk/vim-monokai'
```

##### Vundle (https://github.com/gmarik/Vundle.vim)

```
Plugin 'mnpk/vim-monokai'
```
